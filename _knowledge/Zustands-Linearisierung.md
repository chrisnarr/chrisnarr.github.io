> [!idea] Idea
>
> - Bestimmung einer **nichtlinearen Transformation**, die das System in die nichtlineare Regelungsnormalform überführt
> - Systemdifferentialgleichungen werden in **Regelungsnormalform** transformiert, aus der das linearisierende Regelgesetz abgeleitet wird
> - Transformationsbeziehung geht aus der Lösung eines Systems von Differentialgleichungen hervor
> - Für Regelungsnormalform lässt sich sehr einfach ein Regelgesetz bestimmen, das das System in Regelungsnormalform linearisiert

> [!pro] **PRO**
>
> - keine resultierende unsichtbare interne Dynamik
