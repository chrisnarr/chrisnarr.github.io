> [!definition] Definition
> A set $\mathcal{R}$ is called a Probabilistic Reachable Set (PRS) at probability level $p$ for the discrete-time system $x_{k+1} = f(x_k,w_k)$ if
> $$x_{0} = 0 \Rightarrow \boldsymbol{Pr}(x_{k} \in \mathcal{R}) \geq p$$
> for all $k \in \mathcal{I}_{\geq0}$.
