> [!definition] Simplified Nyquist Criterion
>
> - If the vector encircles the point (-1, 0) in a clockwise direction, the system is **unstable**
> - If the vector does not encircle this point at all, the system is **stable**
> - If the vector encircles the point (-1, 0) in a counterclockwise direction, the system is **marginally stable**
