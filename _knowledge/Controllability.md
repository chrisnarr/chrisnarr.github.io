> [!idea] Idea
> There exists some system input $u$ that can be used to move the state of a system from **any** initial state to **any** final state in **finite time** >**Note:** The state must only be reached, not maintained

> [!definition] Definition: Controllability
> A system of order $n$ is called **controllable**, if the controllability matrix $S_C$ has rank $n$ with
>
> $$
> S_C = \begin{pmatrix} B & AB & A^2B & \dots & A^{n-1}B \end{pmatrix}
> $$
