> [!idea] Idea
> A compact set is a set that is simultaneously **bounded** and **closed**, meaning the set can be contained inside boundaries and it contains those boundaries.

> [!example]- Examples
>
> - The interval $(-\infty,-2]$ is not compact because it is not bounded (but closed).
> - The interval $(-1,1)$ is not compact because it is not closed (but bounded).
> - The interval $[0,1]$ is compact because it is both closed and bounded.
